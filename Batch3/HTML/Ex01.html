What is HTML?

HTML is a Text based Markup language used to create Web Pages to develop Web Apps. HTML is simple and easy to develop. It is not a programming language, it is content and is formatted with tags. These tags mark the content with pre defined format of presentation, hense the name Hyper Text mark up Language

HTML is Open source and is maintained by W3C, a consortium of Web Development leaders who define the principles, standards of Web App technologies.

New Tags of HTML 5

Old Web Pages used div tag for all the divisions and blocks within the HTML Document. Programmers were finding it difficult to understand the purpose of div in a certain area of the doc. To Organize the content, HTML5 gave more tags which behaved like div but names were different, to have an intention of placing it different areas.

  • section
  • article
  • header
  • footer
  • nav
  • aside
